1. Был установлен git с официального сайта.

2. Первоначальная настройка была проведена с помощью команд: 
//установка имени и электронной почты
- $ git config --global user.name "belkarina"
- $ git config --global user.email "katya.filosofova@mail.ru"

// параметры установки окончаний строк
- $ git config --global core.autocrlf true
- $ git config --global core.safecrlf warn

// установка отображения unicode
- $ git config --global core.quotepath off

3. Был создан каталог для выполнения лабораторной работы "LR1"

4. В каталоге был инициализирован git командой
- $ git init

5. В каталоге проекта был создан текстовый файл отчёта "FilosofovaEkaterinaNikolaevna_LR1.txt" и проиндексирован командой
- $ git add /слишком длинное фио_LR1.txt/

6. Был создан первый коммит командой
- $ git commit -m "First commit"

7. Отчёт пишется с начала выполнения текущей лабораторной работы. Отчёт был сохранён ещё раз.

8. Известной командой файл отчёта был проиндексирован ещё раз.

9. Был создан второй коммит командой
- $ git commit -m "Second commit"
   Было установлено, что в файле прибавилось 9 изменений, т.е. было добавлено 9 строк после предыдущего коммита. 

10. Отчёт продолжает писаться. 

11. а) Был добавлен файл filewhosemustbeignored.txt и .gitignore. В файл .gitignore была добавлена строка filewhosemustbeignored.txt.
       Данный файл игнорируется git'ом.

    б) Произведено сравнение изменений в исходном файле командой(сравнение между последним коммитом и последним сохранением файла)
- $ git diff
       Для проиндексированного файла используется
- $ git diff --cached
       В консоли появился текст, который был добавлен в файл после последнего коммита.

    в) Был создан и проиндексирован файл filewhosemustbedeleted.txt. Командой
- $ git rm -f filewhosemustbedeleted.txt
       файл был удалён.
    г) В этом же каталоге была создана новая директория newdir. Был создан и проиндексирован файл filewhosemustbemoved.txt. Командой
- $ git mv filewhosemustbemoved.txt newdir/

12. Командами 
- $ git log
- $ git log -p
    была просмотрена информация об истории коммитов и изменении коммитов соответственно.
13. Отмена сделанных изменений была вызвана командой
- $ git revert -m 1 HEAD
