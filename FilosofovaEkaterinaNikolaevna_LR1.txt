1. Был установлен git с официального сайта.

2. Первоначальная настройка была проведена с помощью команд: 
//установка имени и электронной почты
- $ git config --global user.name "belkarina"
- $ git config --global user.email "katya.filosofova@mail.ru"

// параметры установки окончаний строк
- $ git config --global core.autocrlf true
- $ git config --global core.safecrlf warn

// установка отображения unicode
- $ git config --global core.quotepath off

3. Был создан каталог для выполнения лабораторной работы "LR1"

4. В каталоге был инициализирован git командой
- $ git init
